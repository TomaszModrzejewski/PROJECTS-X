Your Guide to Setting Up a New VPS

Follow these steps to get your server up and running securely.

Step 1: Choose a Provider and Deploy a Server

Select a VPS provider (like DigitalOcean, Linode, Vultr, or AWS Lightsail). Choose a server size and an operating system (Ubuntu 22.04 LTS is a great choice). Once deployed, you will receive an IP address for your server.

Step 2: Initial Server Connection

Connect to your server for the first time using SSH. You will typically log in as the 'root' user.

ssh root@YOUR_SERVER_IP

Step 3: Create a New User and Grant Privileges

It's a security best practice to not use the root user for everyday tasks. Create a new user and give them administrative (sudo) privileges.

adduser your_new_user
usermod -aG sudo your_new_user

Step 4: Basic Security - Firewall Setup

Set up a basic firewall to protect your server. UFW (Uncomplicated Firewall) is easy to use. Allow SSH, HTTP, and HTTPS traffic.

ufw allow OpenSSH
ufw allow 'Nginx Full'
ufw enable
